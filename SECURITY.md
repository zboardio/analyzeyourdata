# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

- **Email:** security@zboardio.com
- **GitHub:** Use the [Private Vulnerability Reporting](https://github.com/zboardio/analyzeyourdata/security/advisories/new) feature

Please do **not** open a public issue for security vulnerabilities.

We aim to respond within 48 hours and provide a fix or mitigation plan within 7 days.
