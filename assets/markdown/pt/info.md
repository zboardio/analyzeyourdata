Carregue seus dados, explore-os interativamente e crie visualizações — tudo no seu navegador. Nenhum dado é armazenado no servidor.

> **Atualizações de gráficos em tempo real** — filtre ou agrupe dados na grade e todos os gráficos se atualizam instantaneamente. Este é o poder central da ferramenta.

### Fontes de Dados Suportadas

**Carregamento Direto:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Fontes Externas:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable através de Personal Access Token

### Funcionalidades Principais

- **AG Grid Interativo** — ordene, filtre, agrupe e pivoteie seus dados com ferramentas de nível empresarial
- **Até 3 gráficos independentes** — escolha entre {{VALUE_CHART_TYPE_SUPPORTED}} tipos de gráficos
- **Inteligência de data e hora** — geração automática de colunas baseadas em tempo a partir dos seus dados de datetime
- **Exportação de gráficos** — baixe gráficos individuais ou dashboards combinados como arquivos HTML interativos

> **Exporte dados da grade** — clique com o botão direito na tabela AG Grid para exportar os dados atualmente filtrados e estruturados diretamente para arquivo **CSV ou Excel**. O que você vê na grade é exatamente o que você obtém na exportação.
