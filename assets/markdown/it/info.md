Carica i tuoi dati, esplorarli in modo interattivo e crea visualizzazioni — tutto nel tuo browser. Nessun dato viene memorizzato sul server.

> **Aggiornamenti grafici in tempo reale** — filtra o raggruppa i dati nella griglia e tutti i grafici si aggiornano istantaneamente. Questo è il potere centrale dello strumento.

### Sorgenti Dati Supportate

**Caricamento Diretto:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Sorgenti Esterne:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable tramite Personal Access Token

### Funzionalità Principali

- **AG Grid Interattivo** — ordina, filtra, raggruppa e pivot i tuoi dati con strumenti di livello aziendale
- **Fino a 3 grafici indipendenti** — scegli tra {{VALUE_CHART_TYPE_SUPPORTED}} tipi di grafici
- **Intelligenza datetime** — generazione automatica di colonne basate sul tempo dai tuoi dati datetime
- **Esportazione grafici** — scarica singoli grafici o dashboard combinati come file HTML interattivi

> **Esporta dati dalla griglia** — fai clic destro nella tabella AG Grid per esportare i dati attualmente filtrati e strutturati direttamente in file **CSV o Excel**. Ciò che vedi nella griglia è esattamente ciò che ottieni nell'esportazione.
