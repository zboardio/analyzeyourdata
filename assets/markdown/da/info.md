#### Understøttede datakilder

**Direkte upload:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Eksterne kilder:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable over Personal Access Token

#### Nøglefunktioner

- **Interaktivt AG Grid** — sortér, filtrér, gruppér og pivot dine data med værktøjer i virksomhedsklasse
- **Op til 3 uafhængige diagrammer** — vælg mellem {{VALUE_CHART_TYPE_SUPPORTED}} diagramtyper
- **Datetime intelligens** — automatisk generering af tidsbaserede kolonner fra dine datetime-data
- **Diagrameksport** — download individuelle diagrammer eller kombinerede dashboards som interaktive HTML-filer

> **Eksportér data** fra AG Grid-tabellen >> **højreklik i AG Grid-tabellen** for at eksportere de aktuelt filtrerede og strukturerede data direkte til **CSV** eller **Excel**-fil.
