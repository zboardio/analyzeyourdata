#### Ondersteunde gegevensbronnen

**Directe upload:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Externe bronnen:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable over Personal Access Token
