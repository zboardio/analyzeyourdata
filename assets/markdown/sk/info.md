#### Podporované zdroje dát

**Priamy upload:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Externé zdroje:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable cez Personal Access Token
