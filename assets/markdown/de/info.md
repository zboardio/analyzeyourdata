Laden Sie Ihre Daten hoch, erkunden Sie sie interaktiv und erstellen Sie Visualisierungen — alles in Ihrem Browser. Es werden keine Daten auf dem Server gespeichert.

> **Echtzeit-Diagrammaktualisierung** — Filtern oder gruppieren Sie Daten im Grid und alle Diagramme aktualisieren sich sofort. Das ist die zentrale Stärke des Tools.

### Unterstützte Datenquellen

**Direkter Upload:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Externe Quellen:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable über Personal Access Token

### Hauptfunktionen

- **Interaktives AG Grid** — Sortieren, Filtern, Gruppieren und Pivotieren Ihrer Daten mit professionellen Werkzeugen
- **Bis zu 3 unabhängige Diagramme** — Auswahl aus {{VALUE_CHART_TYPE_SUPPORTED}} Diagrammtypen
- **Datetime-Intelligenz** — automatische Generierung zeitbasierter Spalten aus Ihren Datetime-Daten
- **Diagramm-Export** — Einzelne Diagramme oder kombinierte Dashboards als interaktive HTML-Dateien herunterladen

> **Daten aus dem Grid exportieren** — Klicken Sie mit der rechten Maustaste in die AG Grid-Tabelle, um die aktuell gefilterten und strukturierten Daten direkt als **CSV- oder Excel**-Datei zu exportieren. Was Sie im Grid sehen, ist genau das, was Sie im Export erhalten.
