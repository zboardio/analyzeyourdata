Ladda upp din data, utforska den interaktivt och skapa visualiseringar — allt i din webbläsare. Ingen data lagras på servern.

> **Realtidsuppdateringar av diagram** — filtrera eller gruppera data i tabellen och alla diagram uppdateras omedelbart. Detta är verktygets kärnkraft.

### Datakällor som stöds

**Direktuppladdning:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Externa källor:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable över Personal Access Token

### Nyckelfunktioner

- **Interaktiv AG Grid** — sortera, filtrera, gruppera och pivotera din data med professionella verktyg
- **Upp till 3 oberoende diagram** — välj bland {{VALUE_CHART_TYPE_SUPPORTED}} diagramtyper
- **Datetime-intelligens** — automatisk tidbaserad kolumngenerering från din datetime-data
- **Diagramexport** — ladda ner enskilda diagram eller kombinerade instrumentpaneler som interaktiva HTML-filer

> **Exportera data från tabellen** — högerklicka i AG Grid-tabellen för att exportera den aktuellt filtrerade och strukturerade datan direkt till **CSV eller Excel**-fil. Vad du ser i tabellen är exakt vad du får i exporten.
