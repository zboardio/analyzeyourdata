Téléchargez vos données, explorez-les de manière interactive et créez des visualisations — le tout dans votre navigateur. Aucune donnée n'est stockée sur le serveur.

> **Mise à jour des graphiques en temps réel** — filtrez ou regroupez les données dans la grille et tous les graphiques se mettent à jour instantanément. C'est la puissance fondamentale de cet outil.

### Sources de données prises en charge

**Téléchargement direct :**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Sources externes :**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable via Personal Access Token

### Fonctionnalités principales

- **Grille interactive AG Grid** — triez, filtrez, regroupez et pivotez vos données avec des outils de niveau entreprise
- **Jusqu'à 3 graphiques indépendants** — choisissez parmi {{VALUE_CHART_TYPE_SUPPORTED}} types de graphiques
- **Intelligence temporelle** — génération automatique de colonnes temporelles à partir de vos données datetime
- **Export de graphiques** — téléchargez des graphiques individuels ou des tableaux de bord combinés sous forme de fichiers HTML interactifs

> **Exportez les données depuis la grille** — faites un clic droit dans la table AG Grid pour exporter les données actuellement filtrées et structurées directement vers un fichier **CSV ou Excel**. Ce que vous voyez dans la grille est exactement ce que vous obtenez dans l'export.
