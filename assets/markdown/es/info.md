Cargue sus datos, explórelos interactivamente y cree visualizaciones — todo en su navegador. Ningún dato se almacena en el servidor.

> **Actualización de gráficos en tiempo real** — filtre o agrupe datos en la tabla y todos los gráficos se actualizan instantáneamente. Este es el poder principal de la herramienta.

### Fuentes de Datos Compatibles

**Carga Directa:**
- Excel `.xlsx`, `.xls`
- CSV `.csv`, `.txt`, `.log`
- JSON
- Parquet
- HDF5
- SQLite `.db`, `.sqlite`, `.sqlite3`

**Fuentes Externas:**
- Microsoft SharePoint / OneDrive
- Google Sheets
- Airtable mediante Personal Access Token

### Características Principales

- **AG Grid Interactivo** — ordene, filtre, agrupe y pivote sus datos con herramientas de nivel empresarial
- **Hasta 3 gráficos independientes** — elija entre {{VALUE_CHART_TYPE_SUPPORTED}} tipos de gráficos
- **Inteligencia temporal** — generación automática de columnas basadas en tiempo desde sus datos de fecha y hora
- **Exportación de gráficos** — descargue gráficos individuales o dashboards combinados como archivos HTML interactivos

> **Exporte datos desde la tabla** — haga clic derecho en la tabla AG Grid para exportar los datos actualmente filtrados y estructurados directamente a archivo **CSV o Excel**. Lo que ve en la tabla es exactamente lo que obtiene en la exportación.
